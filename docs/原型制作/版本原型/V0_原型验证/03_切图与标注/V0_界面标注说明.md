# V0 界面标注说明

## 标注原则

- 所有页面以 `750 x 1624` 为标准尺寸输出。
- 标注优先写结构尺寸和相对关系，不在原型阶段过度细化像素微调。
- 交互说明要写“点击后去哪里”，而不只写“这是个按钮”。

## 研发优先关注

- 顶部安全区与底部页签区必须预留固定高度。
- 章节页、功法页都采用“上信息 + 中内容 + 下导航”的统一结构。
- 战斗页信息要尽量压缩到上方和下方，中心区域留给战斗表现。
- 三选一弹窗必须一次完整展示 3 张卡，避免横向滚动。

## V0 推荐输出物

- 页面设计稿：默认态
- 页面设计稿：关键交互态
- 组件示例：按钮 / 卡片 / 弹窗 / 标签
- 标注文档：尺寸、层级、交互路径
