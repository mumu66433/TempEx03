剑侠风云 V0 UE/UI 预览

本页用于集中预览 V0 原型验证版本的关键页面与组件样板。正式规范请优先查看同目录下的说明文档，以及 `docs/原型制作/00_共用规范/UI设计规范_v1.md`。

UI 设计规范 V0 信息架构 组件样式板

01 登录界面

承接现有账号登录后的游戏入口，强调“进入游戏”的单一主路径。

02 章节界面

V0 的默认主界面，负责章节选择与底部导航承接。

03 战斗界面

顶部展示章节和血条，底部展示本局构筑，中间留给战斗表现。

04 功法三选一

V0 核心策略弹窗，强调 3 卡平铺和快速选择。

05 功法列表

展示已解锁功法与图鉴状态，V0 保留查看能力，不展开复杂养成。

06 功法详情浮层

用于查看品质、门派、流派、升星效果和适配构筑说明。

07 通关结算

展示奖励和下一步行动，承接章节推进。

08 失败结算

展示失败反馈和重试建议，避免玩家直接流失。
