# V0 UE / UI 说明

本目录是 `V0_原型验证` 的设计稿和交互说明目录，直接服务当前版本开发。

## V0 设计目标

- 验证“章节推进 + 自动战斗 + 功法三选一”是否成立
- 让玩家在 10 分钟内理解主循环
- 保证页面少、层级浅、操作路径清晰

## V0 设计范围

- 登录页
- 章节页
- 战斗页
- 功法三选一弹窗
- 功法列表页
- 功法详情浮层
- 通关结算页
- 失败结算页

## V0 不做

- 多角色选择页
- 装备页
- 法宝页
- 帮派与社交页
- 活动页和商城复杂页

## 目录说明

- `01_流程与结构`：信息架构、页面流转、交互路径
- `02_界面设计`：V0 页面设计稿和状态稿
- `03_切图与标注`：尺寸标注、交互标注、研发落地提示
