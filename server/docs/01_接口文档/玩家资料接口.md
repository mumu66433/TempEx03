# 玩家资料接口

本文档覆盖当前 V0 范围内与玩家资料、章节进度相关的接口。

## 接口概览

| 方法 | 路径 | 说明 |
| --- | --- | --- |
| `GET` | `/api/player/profile` | 按账号读取玩家基础资料与章节进度 |
| `GET` | `/api/player/chapters` | 按账号读取带解锁状态的章节列表 |
| `PATCH` | `/api/player/current-chapter` | 更新当前选中的章节 |

## GET /api/player/profile

### 请求参数

| 字段 | 位置 | 必填 | 说明 |
| --- | --- | --- | --- |
| `account` | query | 是 | 玩家账号 |

### 成功响应

HTTP 状态码：`200`

```json
{
  "ok": true,
  "player": {
    "id": "cm123",

    "nickname": "PABC1234",

    "createdAt": "2026-04-25T02:50:00.000Z",

  },

```

## GET /api/player/chapters

### 请求参数

| 字段 | 位置 | 必填 | 说明 |
| --- | --- | --- | --- |
| `account` | query | 是 | 玩家账号 |

### 成功响应

HTTP 状态码：`200`

```json
{
  "ok": true,
  "currentChapterId": 1,
  "highestUnlockedChapterId": 1,
  "chapters": [
    {
      "id": 1,
      "name": "第一章",
      "unlocked": true,
      "isCurrent": true
    }
  ],
  "count": 1
}
```

## PATCH /api/player/current-chapter

### 请求体

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `account` | `string` | 是 | 玩家账号 |
| `chapterId` | `number` | 是 | 目标章节 ID |

### 成功响应

HTTP 状态码：`200`

返回结构与 `GET /api/player/profile` 一致。

### 错误语义

- `账号不能为空`
- `账号不存在`
- `chapterId 必须是正整数`
- `章节不存在`
- `目标章节尚未解锁`
